# Operators - arithmetic, precedence, comparison, logical and assignment

- arithmetic 
```rust

// sum: +
let sum = 1 + 1;
println!("{:?}", sum);

// subtraction: - 
let sub = 2 - 2;
println!("{:?}", sub);

// multiplication: *
let mul = 3 * 3;
println!("{:?}", mul);

// division: /
let div = 4 / 4;
println!("{:?}", div);

// remainder: %
let rem = 5 % 5;
println!("{:?}", rem);  
```

- arithmetic (extra)
```rust
// print
println!("{:?}", 10 + 10);

// let
let calc: u8 = 2 + 5;
println!("{:?}", calc);

// let + let
let num01: u8 = 4;
let num02: u8 = 5;
println!("{:?}", num01 * num02);

// let ++
let num_01: u16 = 9;
let num_02: u16 = 9;

let result: u16 = num_01 / num_02;
println!("{:?}", result);  
```
